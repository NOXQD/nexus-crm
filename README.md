# Nexus CRM

A dark-themed client management web app built with vanilla HTML, CSS, and JavaScript.

**Live demo:** https://vuunderkind.github.io/nexus-crm/

## Features

- Dashboard with client list, search, and status filters
- Add / edit / delete clients
- Statistics page with charts and KPIs
- Dark SaaS design with glassmorphism and glow effects
- Fully client-side — no backend, data stored in localStorage

## Tech

- Vanilla JS (ES6+, ES Modules)
- CSS custom properties, backdrop-filter, animations
- localStorage for persistence
- No frameworks, no dependencies

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Client dashboard |
| `form.html` | Add / edit client |
| `stats.html` | Analytics & statistics |
| `about.html` | About the project |
