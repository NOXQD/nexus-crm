NexusCRM

Современная CRM-система для малого бизнеса — управляйте клиентами, отслеживайте статусы сделок и анализируйте результаты без лишней сложности.

База клиентов

Храните и систематизируйте всю информацию о клиентах: контакты, статус, заметки и история взаимодействий.

Поиск и фильтрация

Мгновенный поиск по имени, email и телефону. Фильтрация по статусам: Новый, В работе, Завершён.

Статистика

Сводная аналитика: KPI-метрики, разбивка по статусам, прогресс-бары конверсии и график активности.

Редактирование

Изменяйте статус прямо из карточки или открывайте полную форму редактирования для детальных правок.

Без сервера

Все данные хранятся локально в браузере через localStorage. Работает офлайн, без регистрации и облака.

Надёжность

Автовосстановление данных при повреждении localStorage. Демо-данные при первом запуске.

Как пользоваться

1 Перейдите на страницу «Клиенты» — здесь отображаются все клиенты. При первом запуске загружаются демо-данные.
2 Нажмите «Добавить клиента», заполните форму (имя, телефон, email обязательны) и сохраните.
3 Используйте поиск и фильтры по статусу для быстрого нахождения нужных клиентов.
4 Меняйте статус клиента через выпадающий список прямо в карточке — изменения сохраняются мгновенно.
5 Для редактирования нажмите кнопку с иконкой карандаша. Для удаления — кнопку с иконкой корзины (потребуется подтверждение).
6 Страница «Статистика» показывает сводку по базе: KPI-карточки, разбивку по статусам и график активности по месяцам.
